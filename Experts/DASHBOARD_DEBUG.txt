🔍 BƯỚC KIỂM TRA:

BƯỚC 1: Compile Lại EA
═══════════════════════════════════════════════════════════════════
1. Mở MetaEditor (F4)
2. Open: Experts/SMC_ICT_EA.mq5
3. Press F7 để compile
4. Đảm bảo: 0 errors, 0 warnings ✅

BƯỚC 2: Remove EA Cũ
═══════════════════════════════════════════════════════════════════
1. Nếu EA đang chạy trên chart
2. Right-click chart → Expert list → Remove
3. Hoặc kéo EA icon ra khỏi góc chart

BƯỚC 3: Attach EA Mới
═══════════════════════════════════════════════════════════════════
1. Navigator (Ctrl+N)
2. Experts → SMC_ICT_EA
3. Kéo vào chart XAUUSD
4. Settings:
   ✅ InpShowDashboard = true (PHẢI BẬT!)
   ✅ InpShowDebugDraw = true
5. Click OK

BƯỚC 4: Check Log (Ctrl+T → Experts tab)
═══════════════════════════════════════════════════════════════════
Bạn PHẢI thấy:
✅ "📊 Dashboard Settings: ShowDebugDraw=1 ShowDashboard=1"
✅ "✅ Dashboard initialized successfully"
✅ "📊 Test label created - Check top-left of chart!"

Nếu thấy:
⚠️ "Dashboard DISABLED by user settings"
→ InpShowDashboard = false - BẬT LẠI!

BƯỚC 5: Check Chart Properties
═══════════════════════════════════════════════════════════════════
Right-click chart → Properties → Common tab
✅ Show objects descriptors: BẬT (checked)
✅ Show trade levels: BẬT
✅ Allow objects: BẬT

BƯỚC 6: Tìm Test Label
═══════════════════════════════════════════════════════════════════
Ngay sau khi attach EA, bạn sẽ thấy:

Góc TRÊN BÊN TRÁI chart:
┌────────────────────────┐
│ SMC/ICT EA v2.1        │  ← Chữ VÀNG
│ Test Dashboard         │
│ If you see this,       │
│ it works!              │
└────────────────────────┘

NẾU THẤY → Dashboard hoạt động! ✅
NẾU KHÔNG THẤY → Vấn đề với chart/MT5 settings

═══════════════════════════════════════════════════════════════════
 🔧 GIẢI PHÁP NẾU VẪN KHÔNG HIỂN THỊ
═══════════════════════════════════════════════════════════════════

GIẢI PHÁP 1: Check AutoTrading
───────────────────────────────────────────────────────────────
Toolbar MT5:
  ✅ Nút "AutoTrading" phải BẬT (màu xanh)
  
Nếu TẮT → EA không chạy → Không có dashboard

GIẢI PHÁP 2: Check Chart Scale
───────────────────────────────────────────────────────────────
Dashboard ở tọa độ (15, 20) - góc trên trái

Nếu chart bị zoom/scroll:
1. Press Home key → về đầu chart
2. Scroll lên trên cùng
3. Zoom out một chút

GIẢI PHÁP 3: Manual Test
───────────────────────────────────────────────────────────────
Mở Tools → Objects → List (Ctrl+B)
Tìm object: "SMC_TestLabel"

Nếu CÓ → Object đã tạo, vấn đề là vị trí/màu
Nếu KHÔNG → Object không được tạo, check permissions

GIẢI PHÁP 4: Check Terminal Tab
───────────────────────────────────────────────────────────────
Ctrl+T → Tab "Experts"

Xem log:
  - Có "Initialization completed"?
  - Có "Dashboard initialized"?
  - Có errors nào không?

GIẢI PHÁP 5: Force Redraw
───────────────────────────────────────────────────────────────
Trong OnTick(), dashboard được update

Đợi vài giây → Nếu thấy trong log:
  "✅ Dashboard updated | State: SCANNING"
  
→ Dashboard đang chạy nhưng không hiển thị

GIẢI PHÁP 6: Thay Đổi Position
───────────────────────────────────────────────────────────────
Edit draw_debug.mqh line 229:
  
  Thay đổi Y coordinate:
  ObjectSetInteger(0, objName, OBJPROP_YDISTANCE, 50);
                                                   ↑ Thử 50, 100, 150
  
  Compile lại và test
